# Memory Templates

Copy these templates to the appropriate paths and fill in your own data.

| Template | Copy to | Purpose |
|----------|---------|---------|
| `human.template.md` | `08-memory/core/human.md` | Your profile (gitignored) |
| `projects.template.md` | `08-memory/core/projects.md` | Active projects snapshot (gitignored) |
| `working.template.md` | `08-memory/working/<project>.md` | Per-project working memory (gitignored) |

The `core/persona.md`, `core/stack.md`, and all `semantic/` + `archive/` files are tracked in git and serve as working examples.
